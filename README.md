# Quick-Task Tracker

## Build instructions

1. Clone the repo and open it in Visual Studio 2022, and make sure to have the ".NET desktop development" package installed (It'll probably work in VS 2026, but I haven't tested it so ¯\_(ツ)_/¯)
2. To the left of the green "run" button select "Release" from the dropdown(by default it should have "Debug" selected)
3. From the top menubar select "Build -> Build Solution"
4. Wait
5. Run the .exe in "\[Project Folder]\bin\Debug\net8.0-windows" or "\[Project Folder]\bin\Release\net8.0-windows"
6. Enjoy! (or not because it kinda sucks)
